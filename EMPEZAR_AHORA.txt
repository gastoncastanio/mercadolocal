BUENA NOTICIA: La app ahora funciona sin necesidad de OpenAI o MongoDB

Simplemente sigue estos 3 pasos:

═══════════════════════════════════════════════════════════════════════════════

PASO 1: Instalar Node.js (SI NO LO TIENES)

Ve a: https://nodejs.org/
Descarga la versión LTS
Instala y reinicia tu terminal

Verifica que funciona:
  node --version
  npm --version

═══════════════════════════════════════════════════════════════════════════════

PASO 2: Instalar Backend

Abre Terminal y ve a:

  cd /Users/user/Desktop/logoai/backend

Instala las dependencias:

  npm install

(Esto toma 2-3 minutos)

═══════════════════════════════════════════════════════════════════════════════

PASO 3: Instalar Frontend

En la MISMA terminal, ve a:

  cd ../frontend

Instala las dependencias:

  npm install

(Esto toma 3-4 minutos también)

═══════════════════════════════════════════════════════════════════════════════

PASO 4: EJECUTAR LA APP (Abre 2 terminales diferentes)

TERMINAL 1 - Backend:

  cd /Users/user/Desktop/logoai/backend
  npm run dev

Deberías ver:
  ✅ "🚀 Servidor LogoAI escuchando en http://localhost:3001"

TERMINAL 2 - Frontend:

  cd /Users/user/Desktop/logoai/frontend
  npm run dev

Deberías ver:
  ✅ "Local: http://localhost:5173"

═══════════════════════════════════════════════════════════════════════════════

PASO 5: ABRE EN TU NAVEGADOR

Copia y pega en tu navegador:

  http://localhost:5173

¡LISTO! Ya tienes LogoAI funcionando 🎉

❓ PREGUNTAS FRECUENTES:

P: ¿Los logos son reales?
R: No, son demostraciones. Cuando agregues tu OpenAI API Key, generará
   logos REALES con IA.

P: ¿Dónde guarda los proyectos?
R: En memoria mientras la app está corriendo. Sin MongoDB real, no persisten
   cuando cierras.

P: ¿Cómo agrego OpenAI después?
R:
  1. Ve a https://platform.openai.com/api/keys
  2. Crea una clave
  3. Edita backend/.env y pega la clave en OPENAI_API_KEY
  4. Reinicia el backend (Ctrl+C y npm run dev de nuevo)

P: ¿Cómo agrego MongoDB?
R:
  1. Ve a https://www.mongodb.com/cloud/atlas
  2. Crea un cluster gratuito
  3. Obtén la "Connection String"
  4. Pégala en backend/.env como MONGODB_URI
  5. Reinicia el backend

❌ SI TIENES ERRORES:

Error: "npm: command not found"
  → Node.js no está instalado
  → Descarga de https://nodejs.org/

Error: "Port 3001 already in use"
  → Otro programa usa ese puerto
  → Cambia PORT en backend/.env a 3002

Error: "Cannot find module"
  → Las dependencias no se instalaron
  → Ejecuta "npm install" de nuevo
